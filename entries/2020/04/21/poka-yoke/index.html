Poka-Yoke

April 21, 2020

Poka-Yoke is a methodology in which a system is designed to methodologically discourage incorrect operation.

E.g. (from the above Wikipedia page):

a fixture that holds pieces for processing might be modified to only allow pieces to be held in the correct orientation, or a digital counter might track the number of spot welds on each piece to ensure that the worker executes the correct number of welds.

(It was coined/canonicalized by Japanese automobile manufacturing at Toyota)
